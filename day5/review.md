## COLOR

Our eyes have 3 cones that can perceive red, green and blue.

```css

div {
  color: red;
  color: rgb(255, 0, 0);
  color: rgba(255, 0, 0, 0.5); /* 50% alpha color of red */
  color: #FF0000; /* hexadecimal */
}

```
